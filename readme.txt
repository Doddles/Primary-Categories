A plugin to select a primary category for each post

== Description ==

Primary Category enables the selection of a Primary Category for each post. The primary category can then be viewed as a standard archive page - for example: https://example.com/primary-category/uncategorized

== Installation ==

= Manual Installation =

1. Upload the entire `/primary-category` directory to the `/wp-content/plugins/` directory.
2. Activate Primary Category through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

How can I access the primary category within the WordPress loop?

We have created a few template tag utilities to call the primary category
